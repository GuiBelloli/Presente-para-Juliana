#  Romantic Page Generator
Gerador de páginas românticas interativas em Python + HTML/CSS/JS.

## 🚀 Como usar
1. Edite `config.json` com seus dados
2. Coloque fotos em `assets/photos/`
3. Execute: `python generator.py`
4. O site pronto sai em `assets/output/index.html`

## 🛠️ Tech Stack
Python, Jinja2, Pillow, QRCode, HTML5, CSS3, Vanilla JS
